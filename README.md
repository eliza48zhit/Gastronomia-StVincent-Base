🇻🇨 Gastronomia de San Vicente y las Granadinas - Smart Contract (Base Mainnet)
Este es el vigesimosexto contrato inteligente de mi coleccion profesional "Sabores de America". San Vicente es famosa por sus suelos volcanicos fertiles donde prospera la fruta del pan (breadfruit), un regalo botanico que define su dieta nacional. Este contrato introduce el campo de Metodo de Coccion, documentando como esta fruta se transforma de un alimento basico a un manjar mediante tecnicas tradicionales.

🛠️ Detalles Tecnicos: Sabores Volcanicos On-Chain

Este despliegue aplica estandares de desarrollo robustos en Solidity:

Hito de la Serie: Vigesimosexto bloque completado. Se utiliza la variable metodoCoccion para clasificar la tecnica aplicada (ej. Asado, Hervido, Frito), un factor determinante en la textura de la fruta del pan.

Integridad del Codigo: Arquitectura lineal diseñada para evitar errores de desbordamiento y optimizar el uso de gas en la red Base.

Cero Caracteres Especiales: Codigo limpio de caracteres Unicode para garantizar una lectura tecnica impecable y evitar el ParserError.

Licencia Abierta: Implementado bajo SPDX-MIT.

📖 Modo de Uso

Consultar el Breadfruit: En la pestaña Read Contract, usa consultarPlato (ID 1) para verificar el metodo "Asado" registrado en el constructor.

Registrar Tradicion: En la pestaña Write Contract, usa registrarPlato para añadir platos como "Breadfruit Oil Down" (Coccion: Hervido) o "Breadfruit Puffs" (Coccion: Frito).

Votacion Inmutable: Sistema de Likes y Dislikes para medir la popularidad de la gastronomia de San Vicente en la blockchain.

🐈 Nota de Autoria
🐈 Curiosidad del Gato: En San Vicente, el gato sabe que nada supera el aroma de la fruta del pan asandose sobre las brasas. Mi codigo vigila que los datos se asen con la misma paciencia para que queden grabados para siempre en Base. Mi mision es documentar cada sabor de America, paso a paso. 🐾
